# My Portfolio Website 
#Prism. 

A responsive, multi-page portfolio website built using HTML5, CSS3, and JavaScript.
## Pages

| File | Description |
| `index.html` | Home page with hero section and quick navigation cards |
| `about.html` | About me, skills, and education timeline |
| `projects.html` | Project showcase with filter tabs |
| `contact.html` | Contact form and social links |
| `style.css` | Shared stylesheet for all pages |
| `script.js` | Shared JavaScript (scroll reveal, mobile menu) |

## Features

- Fully responsive design using CSS Flexbox and Grid
- Media queries for mobile, tablet, and desktop
- Smooth scroll-reveal animations using Intersection Observer
- Project filter tabs (All / HTML / CSS / JavaScript)
- Fixed navigation with active page highlighting
- Mobile hamburger menu
- Dark theme with gradient accents

## Technologies Used

- HTML5 (semantic tags, boilerplate structure)
- CSS3 (custom properties, animations, transitions, media queries)
- Vanilla JavaScript (DOM manipulation, Intersection Observer)
